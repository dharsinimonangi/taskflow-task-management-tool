# TaskFlow - Task Management Tool

TaskFlow is a modern full-stack Task Management Tool inspired by Trello and Asana.

## Features

- User Authentication (Login/Register)
- Dashboard with task statistics
- Create Tasks
- Edit Tasks
- Delete Tasks
- Kanban Board
- Task Calendar
- Dark/Light Theme
- Search Tasks
- Completed Tasks Tracking
- Responsive UI
- Separate tasks for different users

## Tech Stack

### Frontend
- React.js
- React Router DOM
- Context API
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Folder Structure

```bash
backend/
frontend/
```

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
